# ToDoList

### Опис ідеї MVP
Цей проєкт є мінімально життєздатним продуктом (MVP) для керування щоденними завданнями. Він дозволяє користувачам створювати список справ, відстежувати їх виконання та видаляти завершені задачі, що допомагає ефективно організовувати робочий час.
